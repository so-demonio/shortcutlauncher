Shortcut Launcher

An NVDA add-on for quickly launching programs, folders, and URLs with keyboard shortcuts.

Features

  • Add, edit, and delete shortcuts for programs, folders, and URLs
  • Filter shortcuts by type (All, Programs, Folders, URLs)
  • Remembers your last selected filter
  • Choose default browser for opening URLs

Usage

Press NVDA+Shift+H to open Shortcut Launcher.

Keyboard Shortcuts

  • NVDA+Shift+H - Open Shortcut Launcher
  • Enter - Run selected shortcut
  • Delete - Delete selected shortcut
  • Escape - Close dialog

Settings

Access settings via NVDA Menu → Preferences → Settings → Shortcut Launcher

  • Select default browser for URLs
  • View total shortcut count
  • Quick access to manage shortcuts

Data Storage

Shortcuts are saved in: %APPDATA%\nvda\shortcutLauncher\shortcuts.json

Compatibility

  • Minimum NVDA version: 2023.1
  • Tested with: NVDA 2025.3.2
