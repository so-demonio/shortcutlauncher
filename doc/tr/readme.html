Shortcut Launcher

Programları, klasörleri ve URL'leri klavye kısayollarıyla hızlıca başlatmak için bir NVDA eklentisi.

Özellikler

  • Program, klasör ve URL kısayolları ekleyin, düzenleyin ve silin
  • Kısayolları türe göre filtreleyin (Tümü, Programlar, Klasörler, URL'ler)
  • Son seçilen filtreyi hatırlar
  • URL'ler için varsayılan tarayıcı seçin

Kullanım

Shortcut Launcher'ı açmak için NVDA+Shift+H tuşlarına basın.

Klavye Kısayolları

  • NVDA+Shift+H - Shortcut Launcher'ı Aç
  • Enter - Seçili kısayolu çalıştır
  • Delete - Seçili kısayolu sil
  • Escape - Diyalogu kapat

Ayarlar

Ayarlara erişmek için: NVDA Menüsü → Tercihler → Ayarlar → Shortcut Launcher

  • URL'ler için varsayılan tarayıcı seçin
  • Toplam kısayol sayısını görüntüleyin
  • Kısayolları yönetmeye hızlı erişim

Veri Depolama

Kısayollar şurada kaydedilir: %APPDATA%\nvda\shortcutLauncher\shortcuts.json

Uyumluluk

  • Minimum NVDA sürümü: 2023.1
  • Test edilen sürüm: NVDA 2025.3.2
